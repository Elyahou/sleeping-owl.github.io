Обзор 

Column::{type}('{field name}', '{column label}')

Создает новый столбец и добавляет его к текущей конфигурации модели.

  • string
  • lists
  • count
  • image
  • date
  • action

Зарегистрировать свой тип столбца

Смотрите произвольные столбцы.

Отключить сортировку по столбцу

Column::string('my_field')->sortable(false)

Пометить столбец как сортировку по умолчанию

Column::string('title')->sortableDefault()
