Установка

  1. Добавьте данный пакет в ваш composer.json и запустите composer update (или выполните composer require sleeping-owl/admin:1.x):

    "sleeping-owl/admin": "1.*"

  2. После обновления composer, добавьте service provider в ваш app/config/app.php

    'SleepingOwl\Admin\AdminServiceProvider',

  3. Добавьте это в настройку фасадов app/config/app.php:

    'Admin'                => 'SleepingOwl\Admin\Admin',
'AdminAuth'            => 'SleepingOwl\AdminAuth\Facades\AdminAuth',
'AssetManager'         => 'SleepingOwl\Admin\AssetManager\AssetManager',
'Column'               => 'SleepingOwl\Admin\Columns\Column',
'FormItem'             => 'SleepingOwl\Admin\Models\Form\FormItem',
'ModelItem'            => 'SleepingOwl\Admin\Models\ModelItem',

  4. Запустите эту команду в терминале (если вы хотите узнать, что конкретно делает эта команда, посмотрите документацию по команде install):

    $ php artisan admin:install
  5. Готово! Теперь зайдите по адресу <your_site_url>/admin и авторизуйтесь при помощи данных по умолчанию admin / SleepingOwl.
