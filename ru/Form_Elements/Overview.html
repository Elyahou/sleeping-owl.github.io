Обзор 

FormItem::{type}('{field name}', '{label}')

Создает новый элемент формы и добавляет его к текущей конфигурации модели.

Поддерживаемые типы

  • text
  • textAddon
  • checkbox
  • date
  • time
  • timestamp
  • file
  • image
  • select
  • multiSelect
  • textarea
  • ckeditor
  • view

Валидация

FormItem::text('title')->required()->unique()->validationRule('my-custom-rule')

Подробнее смотрите в разделе валидация.

Произвольные HTML аттрибуты

Вы можете добавить произвольные html аттрибуты к элементам формы.

FormItem::text('title')->attributes(['class' => 'my-class', 'my-attribute' => 1])

Значение по умолчанию

Вы можете указать значение по умолчанию для элемента формы.

FormItem::text('title')->default('My new item title')
