SleepingOwl Admin is an administrative interface builder for your Laravel models.

Features

  • Laravel 4.1, 4.2 and 5.0 support
  • Easy installation and initial configuration
  • Customizable menu
  • Separate authentication
  • Validation support
  • Localization support
  • Model configuration autogenerate artisan command
  • Easy add support for image and file fields to your models
  • RESTful-style routes
  • Customizable filters for models
  • Customizable columns in model table view
  • Customizable forms for models
